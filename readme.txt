== Description ==

Instantempo contest WT show the tweets for official hashtag "#instantempo" (color and size customizable), your actual position, the actual podium and phrases about "tempo". Very customizable.

Simply activate the plugin, and the settings as needed, add widget, then your blog is ready to show Instantempo information.

More info:

* Official Website of the plugin [instantempo WT](http://instantempo.webandtech.it/)
* [Instantempo english support](http://instantempo.webandtech.it/blog/instantempo-seo-contest-wp-plugin/instantempo-wp-english-support)
* [Instantempo supporto in italiano](http://instantempo.webandtech.it/blog/instantempo-seo-contest-wp-plugin/instantempo-wp-supporto-in-italiano)
* More info on [instantempo WT seo contest WordPress plugin](http://instantempo.webandtech.it/wordpress-plugin/).

= Features =

* Options to choose language EN/IT
* Options to show/hide tweets widget
* Options to change layout tweets widget
* Options to show/hide your current position
* Options to show current podium
* Options to show/hide thank link
* Options to show/hide phrases about "tempo"

* Easily installation and customisation

== Installation ==

Follow the steps below to install the plugin.

1. Upload the Instantempo WT seo contest directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/Instantempo WT to configure the button
